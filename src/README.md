<h2 id="tym-wc-table-view"></h2>

> ## 簡易テーブル表示 `[tym-wc-table-view]`

<br>

単純な二次元配列を，簡易にテーブル表示します。
カラムのリサイズが可能です。

<br>

## Usage

```html
<script type="module">
  import 'https://unpkg.com/tym-wc-parts@0.7.1/dist/tym-wc-table-view.umd.js';
</script>

<tym-wc-table-view></tym-wc-table-view>
```
